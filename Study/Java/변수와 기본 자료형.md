변수(Variable)은 값을 저장할 때 사용하며, Java에는 아래 4가지 변수 종류가 있다.
* 지역 변수(Local variables) : 중괄호 내 선언된 변수
* 매개 변수(Parameters variables) : 메소드에 넘겨주는 변수
* 인스턴스 변수(Intance variables) : 메소드 밖에, 클래스 안에 선언된 변수, static 키워드가 없어야 함
* 클래스 변수(Class variable) : 인스턴스 변수 중 타입 선언 앞에 static 키워드가 있는 변수
```java
public class VariableType {

int instanceVariable; // Instance variable

static int staticVariable; // Static variable

  

public void method(int parameterVariable) { // Parameter variable

int localVariable; // Local variable

}

}
```

각 변수는 고유의 생명주기를 가진다. 변수의 종류에 따라 그 생명주기가 다르다. 각 변수의 생명주기는 아래와 같다.
* 지역 변수 : 지역변수를 선언한 중괄호 안에서 유효
* 매개 변수 : 메소드 호출 시 시작되며 메소드 종류 시 소멸
* 인스턴스 변수 : 객체 생성 시 시작되며 객체를 참조하는 다른 객체가 없으면 소멸(가비지 컬렉터 회수)
* 클래스 변수 : 클래스 처음 호출 시 시작되며 자바 프로그램 종료 시 소멸


자바의 자료형은 크게 기본 자료형(Primitive data type)과 참조 자료형(Reference data type)으로 나뉜다.
기본 자료형의 경우 개발자가 새롭게 만들 수 없으며 기본 제공된다. 참조 자료형의 경우 개발자가 생성할 수 있다.
자바에서는 new 키워드를 통해 초기화하는 것을 참조 자료형(String 예외), 그렇지 않고 초기화 가능한 자료형을 기본 자료형이라 한다.

기본 자료형 종류는 아래와 같은 구조로 되어있다. 크게 숫자와 boolean 타입으로 나뉘며, 숫자는 정수형과 소수형으로 나뉜다.
* 자바 자료형
	* 기본 자료형
		* 숫자
			* 정수형
			* 소수형
		* boolean
	* 참조 자료형

위 구조에 맞게 기본 자료형을 나열하면 아래와 같다.
* 정수형 : byte, short, int, long, char
* 소수형 : float, double
* 기타 : boolean

정수형은 해당 타입의 범위가 정해진다.(bsil fd 순서)
여기에서 타입의 범위란 각 자료형이 표현할 수 있는 숫자의 크기 제한을 의미한다. 각 정수형 타입별 숫자 범위는 아래와 같다.


## 기본 자료형

### Byte 타입
* byte는 -128~127까지의 값을 가짐
* 이론상 8bit 모든 공간을 1로 채운다면 -127이 되어야 함
* 하지만 하나의 값이라도 더 제공하기 위해 맨 앞 bit는 0이면 양수, 1이면 음수
* 이로 인해 앞이 1이면서 모두 0인 byte는 -128임(앞이 1이면서 모두 0인 byte만 0)
* 그래서 -128~127

* 최소값인 -128 에 -1을 하면 0111_1111 이 됨, 즉 최댓값 127이 됨
* 반대로 최대값 127에 +1을 하면 1000_0000이 됨, 즉 최솟값 128이됨


### Long 타입
* 선언 시 L을 붙여줘야 int로 인식안하고 Long으로 인식함
```java
long temp = 2L; // long으로 인식
long temp2 = 21131241241244124124 // 컴파일 에러(int로 인식했기 때문)
long temp3 = 21131241241244124124L // long으로 정상 인식
```
