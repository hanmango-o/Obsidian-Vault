* java에서 배열은 아래와 같이 선언
```
int[] array;
int array[];
```
* 배열 변수 정의 시 아무것도 안써주고 대괄호만
* 보통 첫번째 방식의 선언이 선호됨
* 배열 초기화는 아래와 같이
```
int[] array = new int[7];
```
* new를 써주고 타입이름 명시, 대괄호에는 크기 명시
* 배열은 참조 자료형이므로 신규 생성 시 new를 붙여야 함
* ArrayIndexOutOfBoundsException은 배열의 범위를 벗어난 위치에 값을 할당하거나, 값을 참조할 경우 발생
* 모든 기본 자료형과 참조 자료형은 배열로 만들수있음
* 기본자료형으로 만든 배열의 초기값은 아래와 같음
* byteArray : 0
* shortArray : 0
* int : 0
* long = 0
* float = 0.0
* double = 0.0
* char = '₩u0000'
* bool = false
* 참조 자료형을 초기화하지 않고 사용하면 null로 기본값이 됨
* 배열을 그냥 출력하면 아래와 비슷하게 나옴
* `[Ljava.lang.String;@12ef1wdq1`
* 위 값으로 배열을 유추가능
* 가장 값에 `[` 은 객체가 배열이라는 뜻
* L 은 해당 배열은 참조 자료형이라는 뜻
* java.lang.String은 해당 배열이 어떤 타입인지
* @부터는 고유 번호(참조 자료형에서는 toString 메소드를 안만드렁주면 이렇게 나옴)
* 기본 자료형의 배열을 그냐야 출력하면 아래왁 ㅏㅌ음
* byte = `[B@고유번호`
* short = `[S`
* int = `[I`
* ....
* long = `[J`
* boolean = `[Z`
* long과 boolean의 경우 각각 L과 B가 아닌 J 와 Z로 표현됨(참조자료형, Byte가 해당 값을 사용하고 있기 때문)
