변수(Variable)은 값을 저장할 때 사용하며, Java에는 4가지 변수 종류가 있다.
* 지역 변수(Local variables) : 중괄호 내 선언된 변수
* 매개 변수(Parameters variables) : 메소드에 넘겨주는 변수
* 인스턴스 변수(Intance variables) : 메소드 밖에, 클래스 안에 선언된 변수, static 키워드가 없어야 함
* 클래스 변수(Class variable) : 인스턴스 변수 중 타입 선언 앞에 static 키워드가 있는 변수
```java
public class VariableType {

int instanceVariable; // Instance variable

static int staticVariable; // Static variable

  

public void method(int parameterVariable) { // Parameter variable

int localVariable; // Local variable

}

}
```
